IINTS: Open-Source Insulin Management

The foundations of affordable and accessible insulin delivery.

User-Centric Interface & Control

Built on the Raspberry Pi Pico (RP2040) architecture, IINTS combines hardware reliability with an accessible interface. The system features an OLED/TFT display and physical controls, allowing user settings and dosages to be managed directly and transparently via MicroPython configurations.

Modular Physical Design

Built for reproducibility and adaptability. The hardware architecture utilizes 3D-printed components and standard parts, allowing researchers to easily modify the chassis for specific experimental needs.

Clinical Calculation Models

IINTS integrates essential physiological parameters for personalized treatment. The framework facilitates precise calculations based on:

  • ICR: Insulin-to-Carb Ratio
  • ISF: Insulin Sensitivity Factor
  • TDD: Total Daily Dose

Safety & Open-Source Validation

Safety is at the core of the design. IINTS incorporates fundamental fail-safes to minimize operational risks. As a fully open-source project (MIT License), IINTS invites global collaboration and transparent validation.
